# Upload steps for GitHub Pages

1. Open your repository: https://github.com/wuweiniuniu/videos
2. Upload `index.html`, `style.css`, `.nojekyll`, and the `assets` folder to the root of the repository.
3. Put your MP4 video at `assets/demo.mp4`.
4. In GitHub repository settings, open Pages, choose `Deploy from a branch`, select `main` and `/(root)`, then Save.
5. The final project-page address is usually: https://wuweiniuniu.github.io/videos/

If you want the root address `https://wuweiniuniu.github.io/`, create or use a repository named `wuweiniuniu.github.io` instead of `videos`.
