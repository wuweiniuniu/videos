# Hierarchical Adaptive Stability–Energy Control for Distributed-Drive Electric Vehicles on Continuous Curved Roads

This repository provides the source code, dataset files, and demonstration videos for the project:

**Hierarchical Adaptive Stability–Energy Control for Distributed-Drive Electric Vehicles on Continuous Curved Roads**

## Code

The `code` folder contains the MATLAB scripts and Simulink files used in this project.

In particular, `PGA_MPC_MODEL.slx` is the Simulink controller model of the proposed PGA-MPC-based control method.

## Dataset

The `dataset` folder contains the data files used for controller analysis and performance evaluation.

## Demo Videos

The demonstration videos are stored in GitHub Releases due to their large file sizes.

* `mild_condition.mp4`: demonstration under the mild curved-road condition.
* `extreme_condition.mp4`: demonstration under the extreme curved-road condition.
